## 2. 디자인 시스템 결정

### 2.1 컬러 팔레트: Steel Authority

01번, 02번 리서치에서 독립적으로 유사한 팔레트를 추천. 교차 검증 완료.

```
Primary:    #1C1B1A  (Warm Black — 배경, 헤더)
Secondary:  #B08D57  (Champagne Gold — CTA, 강조)
            또는 #d9b061 (밝은 골드 변형)
Background: #F2EFEA  (Warm Cream — 본문 배경)
Text:       #FFFFFF  (히어로 텍스트)
            #1C1B1A  (본문 텍스트)
Gray:       #6B7280  (보조 텍스트)
```

**리뷰 의견**: Steel Authority 팔레트 채택. 철강(다크) + 장인정신(골드) + 따뜻함(크림)의 조합이 이중 포지셔닝을 자연스럽게 수용합니다.

### 2.2 타이포그래피

```
한글 헤드라인:  Noto Serif KR (Bold/Black)     — 장인 브랜딩, 권위
한글 본문:      Pretendard (Regular/Medium)    — 가독성, 현대적
영문 헤드라인:  Cormorant Garamond (Bold)      — 전통+장인, x-height 매칭
영문 본문:      Inter (Regular/Medium)         — 스크린 최적화
```

**타입 스케일** (8px 베이스):
- Display: 72px / 80px line-height
- H1: 48px / 56px
- H2: 36px / 44px
- H3: 24px / 32px
- Body: 16px / 28px (한글은 line-height 1.75)
- Caption: 14px / 20px

### 2.3 레이아웃 원칙

- 섹션 간 패딩: 96-128px
- 콘텐츠 최대 폭: 1200px (텍스트 720px)
- 8px 베이스 스페이싱 그리드
- 모바일 퍼스트 반응형 (breakpoints: 640/768/1024/1280px)

## 4. 기술 구현 결정

| 항목 | 결정 | 근거 |
|------|------|------|
| 프레임워크 | Pure HTML + Tailwind CSS CDN | 최대 이식성, 단일 파일 배포 가능 |
| YouTube 임베드 | lite-youtube-embed | 224배 빠른 로딩, LCP 2.5초 이내 목표 |
| 스크롤 애니메이션 | GSAP + ScrollTrigger | 업계 표준, JOHO Architecture 수준 |
| 스무스 스크롤 | Lenis | 프리미엄 브라우징 경험 |
| 아이콘 | Lucide (CDN) | ISC 라이선스, 건축 관련 아이콘 풍부 |
| 이미지 최적화 | WebP + lazy loading | 첫 로드 800KB 이내 목표 |
| 폰트 로딩 | font-display: swap + 동적 서브셋 | Pretendard 동적 서브셋으로 성능 확보 |
