## 4. 한국 특화 UX 패턴

### 4.1 카카오톡 채널 연동 (필수)

- **한국 스마트폰 사용자의 93% 이상**이 카카오톡 사용 (5,200만+ MAU)
- 카카오톡 채널 = 이메일 리스트의 한국판 (오픈율 압도적으로 높음)
- 이메일 마케팅은 한국에서 효과 제한적

**구현 패턴:**
- 플로팅 카카오톡 상담 버튼 (우하단)
- 노란색(#FEE500) 카카오 브랜드 컬러 아이콘
- "카카오톡 상담" 또는 "카톡 문의" 레이블
- 카카오 채널 추가 버튼 (팔로우 유도)
- 카카오 픽셀 트래킹 연동 (코원하우스 사례)

### 4.2 네이버 지도 연동

- 네이버 지도 = 단순 네비게이션 + 비즈니스 검색 + 예약 + 결제 + 쿠폰 통합 플랫폼
- 네이버 블로그 운영 필수 (검색 노출의 핵심)
- 네이버 쇼핑 연동 (해당 시)
- 외부 웹사이트 검색 순위는 네이버 자체 생태계 아래로 밀림

**구현 패턴:**
- 오시는 길 섹션에 네이버 지도 임베드
- 네이버 블로그 링크 (시공 사례/후기 콘텐츠)
- 네이버 예약 연동

### 4.3 전화 CTA (핵심)

- 한국 건설/시공 업계에서 **전화 상담이 1순위 전환 채널**
- 대표 전화번호 헤더 상단 고정 (1577-xxxx 형태 선호)
- 모바일에서 탭 시 바로 전화 연결 (`tel:` 링크)
- "무료 상담" / "무료 견적" CTA 문구

**구현 패턴:**
- 헤더 우측 상단에 전화번호 상시 표시
- 모바일: 하단 고정 바 (전화 + 카카오톡 + 견적문의)
- "무료 견적 상담" / "지금 전화하세요" CTA 버튼

### 4.4 견적/상담 폼

- "무료 견적 문의" 폼이 핵심 전환 요소
- 가이드북/카탈로그 다운로드 (리드 수집) - 코원하우스 사례
- 쇼룸 방문 예약
- 필수 입력: 이름, 연락처(휴대폰), 건축 예정지, 건축 유형, 예산 범위

### 4.5 정보 밀도 (빨리빨리 문화)

- 한국 UX = **정보 밀집형** (서양의 미니멀과 대비)
- "빨리빨리(Palli Palli)" 문화: 여러 페이지 탐색 없이 한 화면에서 정보 파악
- 제품 사양, 가격, 리뷰, 배송 정보, 프로모션 배지 등 **한 화면에 모두 노출**
- 아이콘보다 **명시적 텍스트 레이블** 선호
- 버튼은 기능을 정확히 설명하는 문구 사용

### 4.6 SNS 채널 필수 연동

한국 건설/인테리어 업체의 표준 SNS 구성:
1. **네이버 블로그** (검색 최적화 핵심)
2. **인스타그램** (시공 사진/포트폴리오)
3. **유튜브** (시공 과정/인터뷰 영상)
4. **카카오톡 채널** (실시간 상담)
5. 페이스북 (선택)

## 5. 한국 웹 타이포그래피 표준

### 5.1 추천 폰트 우선순위

#### 1순위: Pretendard (프리텐다드)
- **특징**: Inter + Source Han Sans 기반으로 정제된 system-ui 대체 글꼴
- **장점**: 가독성 우수, 별도 시각 보정 불필요, 깔끔한 영문 조합
- **라이선스**: SIL Open Font License (상업적 사용/수정/재배포 가능)
- **웹 적용**: CDN `https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/static/pretendard.css`
- **다이나믹 서브셋**: 페이지 내 사용 글자만 다운로드 (성능 최적화)
- **가변 폰트**: Variable weight 지원으로 파일 크기 절감
- **9가지 굵기**: Thin ~ Black
- **특수 변형**: Pretendard GOV (공공서비스용), Pretendard JP (일본어), Pretendard Std (라틴)
- **GitHub**: [orioncactus/pretendard](https://github.com/orioncactus/pretendard)

#### 2순위: Noto Sans KR
- **특징**: Google/Adobe 공동 개발, 현대 한글 음절 11,172자 + KS X 1001 기호 커버
- **적용**: `@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@300;400;500;700&display=swap')`
- **장점**: Google Fonts CDN, 광범위한 문자 커버리지
- **주의**: CJK 폰트 특성상 파일 크기가 큼 (서브셋 필수)

#### 3순위: Noto Serif KR (세리프)
- **용도**: 프리미엄/고급 브랜딩, 헤드라인
- **적용**: Google Fonts 제공

### 5.2 폰트 스택 권장

```css
/* 본문 */
font-family: 'Pretendard', 'Noto Sans KR', 'Nanum Gothic', 'Malgun Gothic', sans-serif;

/* 프리미엄 헤드라인 (선택) */
font-family: 'Noto Serif KR', 'Batang', serif;
```

### 5.3 한글 타이포그래피 유의사항

- **줄간격(line-height)**: 한글은 영문 대비 넓은 줄간격 필요 (1.6~1.8 권장)
- **자간(letter-spacing)**: 본문 -0.01em ~ 0em, 헤드라인 -0.02em ~ -0.04em
- **폰트 로딩**: `font-display: swap` 필수 (FOIT 방지)
- **서브셋**: 사용 문자만 로딩하여 Core Web Vitals 최적화
- **크로스 브라우저**: Chrome, Safari, 모바일에서 줄간격 렌더링 차이 테스트 필수
- **폴백 주의**: 한글 폰트 누락 시 중국어 글리프가 표시될 수 있음

## 7. 2025-2026 한국 웹 디자인 트렌드

### 건축/건설 웹사이트에 적용 가능한 트렌드

1. **라이브 타이포그래피**: 스크롤 시 글자 자연 등장, 타이핑 효과
2. **벤토 그리드 레이아웃**: 콘텐츠를 블록/카드 형태로 시각적 정돈
3. **3D/인터랙티브 요소**: 마우스 오버, 스크롤 애니메이션으로 역동성
4. **마이크로 인터랙션**: 작은 동작에 정교한 반응 (완성도 인상)
5. **미니멀 UI**: 불필요한 그래픽 요소 최소화
6. **성능 최적화**: 3초 이상 로딩 시 40% 이탈

## 8. 철목수 랜딩페이지 적용 권장 사항

### 8.1 필수 구현 요소

- [ ] **모바일 퍼스트 디자인** (한국 e-커머스 거래의 75%가 모바일)
- [ ] **카카오톡 채널 상담 버튼** (플로팅, 우하단)
- [ ] **전화 CTA** (헤더 상단 고정 + 모바일 하단 고정바)
- [ ] **무료 견적 상담 폼** (이름, 연락처, 건축예정지, 유형, 예산)
- [ ] **Pretendard 웹폰트** (다이나믹 서브셋)
- [ ] **네이버 지도** (오시는 길 섹션)
- [ ] **시공 사례 포트폴리오 그리드** (사진 중심)
- [ ] **건설업 등록/면허 배지** (하단 Trust 섹션)
- [ ] **사업자등록번호** (푸터)
- [ ] **SNS 링크** (네이버 블로그, 인스타, 유튜브, 카카오)

### 8.2 디자인 방향

- **하이엔드 미니멀**: 건축사무소 수준의 시각적 품질 (JOHO, 로그디자인 참고)
- **정보 밀도 밸런스**: 한국 시장의 정보 밀집 선호와 프리미엄 미니멀 사이 균형
- **풀스크린 히어로**: 스틸하우스 대표 시공 사진/영상
- **시공 과정 스토리텔링**: 설계 -> 골조 -> 마감 프로세스 시각화
- **신뢰 구축 우선**: 인증 배지 + 시공 실적 + 고객 후기를 메인 플로우에 배치

### 8.3 경쟁사 대비 차별화 포인트

기존 스틸하우스/전원주택 업체 웹사이트 대부분이 **구형 기업 홈페이지 템플릿** 수준에 머물러 있음. 철목수는 다음으로 차별화 가능:

1. **건축사무소 수준의 비주얼 퀄리티** (포스코휴먼스의 기업형 vs 건축 포트폴리오형)
2. **모던 인터랙션** (GSAP 스크롤 애니메이션, 마이크로 인터랙션)
3. **스토리 기반 랜딩** (단순 정보 나열이 아닌 건축 철학/과정 서사)
4. **속도**: 3초 이내 로딩 (한국 사용자 기대치)
